# 💱 Projeto 04 — Conversor de Moedas em React

Projeto prático de desenvolvimento web desenvolvido com **React** e **Vite**, focado na criação de uma aplicação dinâmica para conversão de moedas estrangeiras em tempo real, explorando manipulação de estados e consumo de dados.

## ✨ Funcionalidades

- **Conversão Dinâmica:** Componente interativo que permite ao usuário selecionar e calcular a conversão entre diferentes moedas (como USD, EUR, BRL).
- **Gerenciamento de Estado:** Uso estratégico de Hooks do React (`useState`) para capturar os inputs do usuário e atualizar os valores em tempo real.
- **Arquitetura Componentizada:** Separação clara da lógica de conversão em um componente isolado (`ConversorDeMoedas.jsx`) para manter o código limpo e reutilizável.
- **Estilização Modular:** Interface moderna e responsiva com estilizações CSS dedicadas para o ecossistema React.

## 🛠️ Tecnologias Utilizadas

- React (v18+)
- Vite 
- JavaScript (ES6+)
- CSS3
